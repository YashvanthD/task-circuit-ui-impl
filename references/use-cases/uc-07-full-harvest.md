## Business Rules

1. **All fish harvested** - count set from current_count automatically
2. **Stock terminated** - Status changes to "harvested"
3. **Pond freed** - Status changes to "empty", can be restocked
4. **Withdrawal check** - Cannot harvest if under medication withdrawal
5. **Cycle summary** - Auto-generate profitability summary

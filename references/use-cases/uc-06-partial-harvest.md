## Business Rules

1. **Count must be valid** - Cannot harvest more than current_count
2. **Withdrawal check** - Cannot harvest if under medication withdrawal
3. **Stock remains active** - After partial harvest, stock continues
4. **Revenue tracking** - Total value added to stock revenue
5. **Gate pass optional** - Can create gate pass for logistics tracking

---

## Acceptance Criteria

- [ ] Harvest record created with type="partial"
- [ ] Stock.current_count decreases by harvest count
- [ ] Stock.harvest_count increases
- [ ] Stock status remains "active"
- [ ] Activity logs created
- [ ] Cannot harvest if under withdrawal
- [ ] Gate pass created if requested
